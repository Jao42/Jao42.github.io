Suap Scraper

Esse projeto é uma biblioteca para raspar o sistema que é usado no IFPB para... coisas - é muita coisa - nele é que, por exemplo, os professores registram as notas dos alunos(e a gente corre para acessar e ver se passou), e aí é onde essa biblioteca atua atualmente, por enquanto ela só pega as informações do boletim e retorna como um JSON. Fiz esse projeto primeiramente porque o SUAP do IFPB não tem uma API disponível(do IFRN tem, triste) e eu queria uma forma de receber notificações quando tivesse uma nota nova no sistema(consegui!) e futuramente talvez fazer outras automações. Aprendi coisas como requisições assíncronas com httpx e asyncio(cada bimestre é um link, se houver muitas notas no boletim pode demorar horrores de forma síncrona... 30 seg até); a trabalhar com sessões, pois é preciso se autenticar para conseguir as informações, e sem sessões não dá para persistir os cookies de autenticação entre as requisições; além de que aprendi a como enviar uma biblioteca ao pypi para poder ser instalado com o pip! Maneirão! Tem muita coisa ainda para fazer nesse projeto(documentar melhor, fazer testes, raspar mais coisas...) mas já funciona.
